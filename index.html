The issue you were facing with camera movement on a phone is a common problem when using A-Frame with external components that can conflict with the default `look-controls`, which is responsible for using the gyroscope for orientation. This video explains how to add `device-orientation-permission-ui` to your scene, which is often a necessary step to enable gyroscope controls on mobile devices, especially for iOS.
